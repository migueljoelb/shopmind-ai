# Workflow de n8n

Aquí se guardará el archivo `workflow.json` exportado desde n8n
(lo generamos en la Fase 5: Configuración de n8n).

## Cómo importar este workflow en tu propio n8n
1. Abre tu instancia de n8n.
2. Ve a Workflows > Import from File.
3. Selecciona el archivo `workflow.json` de esta carpeta.
